Métodos de cruce de jQuery

# Method Description
1 add() Adds elements to the set of matched elements
2 addBack() Adds the previous set of elements to the current set
3 andSelf() Deprecated in version 1.8. An alias for addBack()
4 children() Returns all direct children of the selected element
5 closest() Returns the first ancestor of the selected element
6 contents() Returns all direct children of the selected element (including text and comment nodes)
7 each() Executes a function for each matched element
8 end() Ends the most recent filtering operation in the current chain, and return the set of matched elements to its previous state
9 eq() Returns an element with a specific index number of the selected elements
10 filter() Reduce the set of matched elements to those that match the selector or pass the function's test
11 find() Returns descendant elements of the selected element
12 first() Returns the first element of the selected elements
13 ehas() Returns all elements that have one or more elements inside of them
14 is() Checks the set of matched elements against a selector/element/jQuery object, and return true if at least one of these elements matches the given arguments
15 last() Returns the last element of the selected elements
16 map() Passes each element in the matched set through a function, producing a new jQuery object containing the return values
17 next() Returns the next sibling element of the selected element
18 nextAll() Returns all next sibling elements of the selected element
19 nextUntil() Returns all next sibling elements between two given arguments
20 not() Returns elements that do not match a certain criteria
21 offsetParent() Returns the first positioned parent element
22 parent() Returns the direct parent element of the selected element
23 parents() Returns all ancestor elements of the selected element
24 parentsUntil() Returns all ancestor elements between two given arguments
25 prev() Returns the previous sibling element of the selected element
26 prevAll() Returns all previous sibling elements of the selected element
27 prevUntil() Returns all previous sibling elements between two given arguments
28 siblings() Returns all sibling elements of the selected element
29 slice() Reduces the set of matched elements to a subset specified by a range of indices
